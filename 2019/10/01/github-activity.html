Comment

なんやかんや色々作った。Ruby しか書いてない。 いまさらだが諸言語に比べて

  • 書きやすい
  • 動かしやすい
  • 静的な型解析なくてもテスト書けば良い
    • 実際問題、 静的解析/コンパイル時間 と テスト書く/実行 の総和比べてどっちが早いかって言うと普通に微妙な所だと思う
    • テストは型合ってても書く必要あるし

ので、まだまだ実用性高いと感じている。この辺の認識についてはいつかまとめて書きたい気もするが、その日は来るのであろうか (反語)。
